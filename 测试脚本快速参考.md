## ❌ 常见错误

### 错误 1: 导入失败
```
✗ 导入失败: No module named 'app'
```
**解决方案:**
```bash
# 确保在项目根目录
cd /Users/libiao/Desktop/github/dmlv4
```

### 错误 2: Kafka 连接失败
```
Kafka 生产者: ✗ 失败
```
**解决方案:**
```bash
# 检查 Kafka 服务
telnet 10.17.154.252 9092

# 检查环境变量
echo $KAFKA_BOOTSTRAP_SERVERS
```

### 错误 3: 认证失败（仅完整测试）
```
✗ 无法获取认证令牌
```
**解决方案:**
```bash
# 设置测试令牌
export TEST_TOKEN=your_token
# 或修改脚本中的 test_user 字典
```
