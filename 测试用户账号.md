## 使用说明

1. **登录系统**：使用上述任一账号登录前端应用（http://localhost:3000）
2. **权限验证**：每个角色看到的界面和可用操作不同
3. **测试场景**：
   - 管理员可以管理所有数据和用户
   - TPM可以创建和管理需求、测试用例
   - TESTER可以创建测试用例和执行工作流
   - AUTOMATION专注自动化和资产管理

## 创建新用户

如需创建新用户，请使用以下命令：

```bash
cd /Users/libiao/Desktop/github/dmlv4/backend
python scripts/create_user.py \
  --user-id <用户ID> \
  --username <用户名> \
  --password '<密码>' \
  --roles <角色1>,<角色2> \
  --email <邮箱>
```

示例：
```bash
python scripts/create_user.py \
  --user-id newuser001 \
  --username 新用户 \
  --password 'NewUser@123' \
  --roles TESTER \
  --email newuser@example.com
```
