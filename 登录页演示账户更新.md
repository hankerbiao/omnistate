# 登录页演示账户更新说明

## 更改概述

已将登录页的演示账户从模拟账户更新为真实的用户账户，支持一键快速登录。

## 更新内容

### 1. 更新了演示账户列表

| 角色 | 用户ID | 用户名 | 密码 | 颜色主题 |
|------|--------|--------|------|----------|
| **管理员** | admin001 | 系统管理员 | Admin@123 | 蓝色渐变 |
| **TPM** | tpm001 | 张三（TPM） | TPM@123 | 琥珀色渐变 |
| **测试工程师** | tester001 | 李四（测试工程师） | Test@123 | 翠绿色渐变 |
| **自动化工程师** | auto001 | 王五（自动化工程师） | Auto@123 | 紫色渐变 |

### 2. 功能特性

- **一键登录**：点击演示账户按钮即可自动填充用户名和密码，并自动执行登录
- **视觉区分**：每个角色使用不同的颜色主题进行区分
- **实时反馈**：按钮点击时有动画效果和悬停反馈
- **提示文字**：添加了"点击上方按钮直接登录体验"的提示

## 代码修改详情

### 1. Login.tsx 更新

#### 接口定义更新
```typescript
// 修改前

```

#### 演示账户按钮更新

### 2. App.tsx 更新

#### handleQuickLogin 函数更新
```typescript
// 修改前
const handleQuickLogin = useCallback((user_id: string) => {
  setLoginForm({ ...loginForm, user_id, password: '123456' });
}, [loginForm]);

// 修改后
const handleQuickLogin = useCallback((user_id: string, password: string) => {
  setLoginForm({ user_id, password, rememberMe: false });
  setLoginError('');
  // 延迟一点时间让表单更新，然后执行登录
  setTimeout(async () => {
    await handleLogin();
  }, 50);
}, [handleLogin]);
```

## 使用方法

1. 打开登录页（http://localhost:3000）
2. 在登录表单下方会看到"🚀 快速体验演示账户"区域
3. 点击对应角色的按钮
4. 系统会自动填充用户名和密码，并执行登录
5. 登录成功后自动跳转到主界面
